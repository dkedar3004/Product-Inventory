Product Inventory Management

Application Contains following features

Search Product

Wherever list of records are viewed, there should be Search filter provided to only display a subset
of records, so that user can quickly see it when there are many records to be viewed.

Responsive UI

have a responsive user interface and make the user interface better, you should use Bootstrap or
Angular Material. Also, make the UI as modern as possible by using Bootstrap Cards, Grid,
Carousel etc. instead of having just a plain table. Appropriate input fields should be used to make it user friendly.

Customozation

While viewing product list, when different fields of the products are displayed, customiz
ation should be provided to display only a subset of the fields.

Test Cases

41 Test Cases Angular Component Class, Component DOM test cases has to be implemented
and the Karma output

Registration and Authentication

Users should be able to register to identify themselves to the system so that they can
sign in and perform add, edit and delete operations. Viewing of product list summary should be all
owed even without signing in.
